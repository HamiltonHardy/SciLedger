Java Source Code

Import .java files using an IDE (such as NetBeans, Eclipse) and view/run from IDE. Main driver is located in DataStorage.java

If compiling from command line, remove any package statements from code files and run:
	$ javac *java
	$ java DataStorage

Optional remove .class files after finished
	$ rm *.class
